Website sistem pakar berbasis Forward Chaining yang dirancang menggunakan bahasa native HTML, CSS JavaScript tanpa framework & library apa pun.

Tentang Sistem

Sistem pakar ini dibuat untuk membantu pengguna mendiagnosa kerusakan komputer secara mandiri. Dengan memasukkan beberapa gejala, sistem akan memberikan analisis awal menggunakan metode Forward Chaining. Perlu diingat bahwa informasi yang diberikan disini hanya diagnosa awal dan tidak 100% benar, untuk kepastian data harap konsultasi ke ahlii yang sebenarnya

Fitur Unggulan

Alat Pembuatan

Di-build hanya dengan native JavaScript, HTML, dan CSS serta github untuk version control systemnya.

UI Simple & Modern

Tampilan sederhana, bersih, dan menggunakan warna-warna tema gelap agar nyaman untuk dilihat dan dioperasikan.

Scroll Animation

Scroll animation memberikan pengalaman interaktif dan lebih hidup. Dimana saat kita mengscroll ke sebuah elemen maka elemen tersebut akan ada entrance masuknya

Forward Chaining

metode penalaran yang dimulai dari fakta atau data yang ada, kemudian mencocokkan fakta tersebut dengan aturan IF-THEN
