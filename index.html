Deteksi Kerusakan Komputer Secara Cerdas & Cepat

Website sistem pakar berbasis Forward Chaining yang dirancang menggunakan bahasa native HTML, CSS JavaScript tanpa framework & library apa pun.

Tentang Sistem

Sistem pakar ini dibuat untuk membantu pengguna mendiagnosa kerusakan komputer secara mandiri. Dengan memasukkan beberapa gejala, sistem akan memberikan analisis awal menggunakan metode Forward Chaining. Cocok digunakan untuk mahasiswa, teknisi pemula, maupun pengguna umum.

Fitur Unggulan

UI Modern & Elegan

Tampilan sederhana, bersih, dan menggunakan warna-warna cerah agar nyaman digunakan.

Animasi Lembut

Scroll animation memberikan pengalaman interaktif dan lebih hidup.

Forward Chaining

Metode pelacakan fakta untuk diagnosa kerusakan komputer secara akurat.

Tanpa Framework

Di-build hanya dengan native JavaScript, HTML, dan CSS.
